Éléments obligatoires de la conception de Canada.ca: Spécifications de Canada.ca

Les pages Web du gouvernement du Canada doivent être facilement identifiables comme faisant partie de Canada.ca. Chaque page doit véhiculer l’image de marque globale de Canada.ca et offrir une expérience uniforme aux visiteurs du site.

Ces éléments obligatoires ont été conçus pour refléter la marque numérique de confiance du gouvernement du Canada : moderne, ouverte, utile et crédible.

En-tête et pied de page

Chaque page du gouvernement du Canada partage un en-tête et un pied de page communs. C’est-à-dire que les parties supérieure et inférieure d’une page doivent avoir le même aspect de page en page.

Obtenez les directives détaillées :

  • Pied de page général

Toutes les institutions et les organisations doivent utiliser ce modèle de page pour leur page d’accueil unique.

  • Modèle de page d’accueil institutionnelle

Autres modèles de page obligatoires

Certains types de pages de Canada.ca utilisent des modèles obligatoires. La cohérence dans ces types de pages aide les gens à trouver et à accomplir des tâches.

  • Page de thème : la page principale pour chaque thème de Canada.ca, uniquement pour les institutions responsables des thèmes
  • Page de sujet : obligatoire pour les 2 premiers niveaux de sujets de l’arborescence thématique de Canada.ca, facultative pour les sujets de niveau inférieur
  • Profil de ministre : doit être utilisé pour les ministres, les chefs d’une institution quasi judiciaire sans lien de dépendance et les secrétaires parlementaires
  • Page de nouvelle : gabarit commun pour les produits de nouvelles comme les communiqués de presse, les documents d’information, les avis aux médias et les discours
